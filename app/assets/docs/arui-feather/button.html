Атрибуты
Атрибут Тип По-умолчанию Описание
text String —

Текст кнопки
icon String —

Иконка кнопки
right-addons String —

Список произвольных элементов в конце компонента
left-addons String —

Список произвольных элементов в начале компонента
view String —

Тип кнопки Варианты: 'default', 'action', 'extra', 'other'.
type String 'button'

Поведение кнопки Варианты: 'button', 'reset', 'submit'.
tag String 'button'

Элемент, которым будет компонент Варианты: 'button', 'span'.
width String —

Управление шириной кнопки. При значении 'available' растягивает кнопку на ширину родителя Варианты: 'default', 'available'.
size String 'm'

Размер компонента Варианты: 's', 'm', 'l', 'xl'.
disabled Boolean —

Управление возможности взаимодействия с компонентом
pseudo Boolean —

Отображение dashed границы кнопки
id String —

Идентификатор компонента в DOM
name String —

Имя компонента в DOM
title String —

Текст всплывающей подсказки
tab-index Number —

Последовательность перехода между контролами при нажатии на Tab
togglable String —

Тип переключателя Варианты: 'check', 'radio'.
checked Boolean —

Отображение кнопки в отмеченном (зажатом) состоянии
theme String —

Тема компонента Варианты: 'alfa-on-color', 'alfa-on-white'.
class String —

Дополнительный класс
Слоты
Слот Описание
text

Текст кнопки
icon

Иконка кнопки
right-addons

Список произвольных элементов в конце компонента
left-addons

Список произвольных элементов в начале компонента
События
Событие Описание
click

Обработчик клика по кнопке
focus

Обработчик фокуса кнопки
blur

Обработчик снятия фокуса кнопки
mouse-enter

Обработчик события наведения курсора на кнопку
mouse-leave

Обработчик события снятия курсора с кнопки
mouse-down

Обработчик события нажатия кнопки мыши в момент, когда курсор над кнопкой
mouse-up

Обработчик события отжатия кнопки мыши в момент, когда курсор над кнопкой
key-down

Обработчик события нажатия на клавишу клавиатуры в момент, когда фокус находится на компоненте
key-up

Обработчик события отжатия на клавишу клавиатуры в момент, когда фокус находится на компоненте
Методы
Метод Описание
getNode()

Возвращает корневой HTMLElement компонента.
focus()

Устанавливает фокус на поле ввода.
blur()

Убирает фокус с поля ввода.
