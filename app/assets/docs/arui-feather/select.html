Атрибуты
Атрибут Тип По-умолчанию Описание
mode String 'check'

Тип выпадающего списка Варианты: 'check', 'radio', 'radio-check'.
width String 'default'

Управление возможностью компонента занимать всю ширину родителя Варианты: 'default', 'available'.
directions Array ['bottom-left', 'bottom-right', 'top-left', 'top-right']

Направления, в которые может открываться попап компонента
placeholder String 'Выберите:'

Подсказка, которая отображается в кнопке раскрывающегося списка, в случае, если ни один из пунктов выбран
disabled Boolean false

Управление возможностью редактирования значения
opened Boolean —

Управление видимостью выпадающего списка
equal-popup-width Boolean false

Ширинa выпадающего списка равна ширине кнопки
value Array —

Список выбранных значений
options Array []

Список вариантов выбора
size String 'm'

Размер компонента Варианты: 's', 'm', 'l', 'xl'.
error String —

Содержание попапа с ошибкой
error-directions Array ['right-center', 'right-top', 'right-bottom', 'bottom-left']

Расположение попапа с ошибкой (в порядке приоритета) относительно точки открытия
theme String —

Тема компонента Варианты: 'alfa-on-color', 'alfa-on-white'.
render-button-content Function —

Кастомный метод рендера содержимого кнопки, принимает на вход: массив элементов типа

Слоты
Слот Описание
error

Содержание попапа с ошибкой
События
Событие Описание

Обработчик фокуса на компоненте
blur

Обработчик потери фокуса компонентом
button-focus

Обработчик фокуса на кнопке
button-blur

Обработчик потери у кнопки
menu-focus

Обработчик фокуса на меню
menu-blur

Обработчик потери фокуса у меню
click

Обработчик клика по кнопке компонента
click-outside

Обработчик клика вне компонента
change

Обработчик изменения значения
key-down

Обработчик нажатия на клавишу
Методы
Метод Описание
focus()

Устанавливает фокус на компонент.
blur()

Убирает фокус с компонента.
scrollTo()

Скроллит страницу до компонента.
