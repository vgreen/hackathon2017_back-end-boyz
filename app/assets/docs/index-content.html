Alfa Blueprint Alfa Blueprint — это система прототипирования на основе технологий HTML, CSS и JavaScript, реализующая стиль Альфа Банк. Для использования системы вам достаточно базовых знаний HTML и CSS.
Как использовать? 1. Скачайте архив с Alfa Blueprint Скачать 0.6.2
2. Разархивируйте
3. ./index.html — базовый шаблон для создания прототипа
4. ./docs/index.html — документация по компонентам
Атрибуты Все компоненты Alfa Blueprint обладают атрибутами.

Атрибуты позволяют настроить внешний вид и логику поведения компонентов. Например, кнопка обладает атрибутами "size" и "view", которые позволяют выбрать размер и внешний вид кнопки.

Об атрибутах каждого из компонентов можно узнать подробнее в документации по нему. 
<a-button size="s" view="default">Обычная маленькая кнопка</a-button>
<a-button size="l" view="extra">Большая кнопка для действия</a-button>
Обычная маленькая

Большая для действия
Слоты Слоты — это механизм, позволяющий размещать ваш контент внутри компонентов Alfa Blueprint. По-умолчанию, почти все компоненты обладают слотом для контента, чтобы разместить в нем что-либо просто укажите содержимое внутри тегов компонента. 
<a-button size="m">
    <span>Текст кнопки</span>
</a-button>
Текст кнопки
У части компонентов есть больше, чем один слот.

Это необходимо, чтобы в компоненте можно было разместить больше одного пользовательского контента.

Например, кнопка обладает слотом, в который можно поставить иконку.

Для этого создан механизм именовых слотов. О том, какие слоты у каждого компонента можно узнать в документации по нему. 
<a-button size="m">
    <a-spin size="m" visible="true" slot="icon"></a-spin>
    <span slot="content">
        Загрузка...
    </span>
</a-button>
Загрузка...
События Часть компонентов Alfa Blueprint обладает механизмом событий. Благодаря ему, вы можете отслеживать взаимодействия компонента с пользователей. 
<a-button size="m" id="click-example-button">Нажми на меня!</a-button>
<script>
    document.getElementById('click-example-button')
        .addEventListener('click', function() {
            alert('Привет!');
        });
</script>
Нажми на меня!
Публичные методы У части компонентов Alfa Blueprint есть публичные методы. Они позволяют выполнить какое-либо действие на компоненте.

Например, програмно поставить фокус на поле ввода.

О публичных методах каждого из компонентов можно узнать подробнее в документации по нему. 
<a-button size="s" id="example-button">Нажми на меня!</a-button>
<a-input size="m" id="example-input"></a-input>
<script>
    document.getElementById('example-button').addEventListener('click', function() {
        var input = document.getElementById('example-input');
        input.focus();
    });
</script>
Нажми на меня!
